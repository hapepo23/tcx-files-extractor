# TCX Files Extractor

![Powered By Lazarus](Powered_By_Lazarus.png) 

-----

This console application, programmed in Free Pascal, extracts all essential activity information from all TCX files in the specified directories. It processes 1600 TCX files in 45 seconds (ThinkPad T14 Gen1, Intel Core i7-10610U CPU).

Usage:

    ./tcx_files_extractor <directory-1> <directory-2> ... <directory-n>
